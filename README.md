# TravelerZ

# General description

TravelerZ is a web application for anybody that wants to easily plan their trips. It provides: 

- weather information
- hotels or accommodations
- nearby restaurants
- nearby landmarks


# Technologies used

Frontend: Angular 13.1.1

Backend: Spring Boot 2.5 / .NET 6.0.2

Database: MySQL / MongoDB
